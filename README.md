# Users-App
Приложение для управления пользователями

## Возможности
- авторизация
- создание пользователя
- редактирование пользователя
- удаление пользователя

## Технологии

-  TypeScript
-  Webpack + Npm
-  React 16
-  Ant design 5 и styled-components
-  React router 6
-  TanStack Query 4 + axios
-  dayjs
- Архитектура FSD (Feature-Sliced Design)

## Установка

git clone https://github.com/Shakhrii/Users-App.git
cd Users-App
npm install

## Запуск

npm run dev
